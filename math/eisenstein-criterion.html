はじめに

Q[X]\mathbb{Q}[X]Q[X] や Z[X]\mathbb{Z}[X]Z[X] の既約元を判定する方法として，次のEisenstein の既約判定法は広く知られています：

Z\mathbb{Z}Z-係数の一変数多項式f(X)=anXn+⋯+a1X+a0∈Z[X]f(X) = a_n X^n + \dots + a_1 X + a_0 \in \mathbb{Z}[X]f(X)=an​Xn+⋯+a1​X+a0​∈Z[X]について，次を満たす素数 ppp が存在するとする：

  1. p∤anp \nmid a_np∤an​

  2. p∣aj(j=0,…,n−1)p \mathrel{\mid} a_j \quad (j = 0, \dots, n-1)p∣aj​(j=0,…,n−1)

  3. p2∤a0p^2 \nmid a_0p2∤a0​

この時，f(X)f(X)f(X) は Z\mathbb{Z}Z および Q\mathbb{Q}Q 上既約である．

実は，Eisenstein の既約判定法は，係数環が一般のUFDやその商体の場合にも使うことが出来ます1．これにより，多変数多項式の既約性を簡単に判定することが出来る場合があります．

UFD 版の証明と応用例

  • u∈Du \in Du∈D が単元 ⇔def\xLeftrightarrow{\mathrm{def}}def​ ある i∈Di \in Di∈D があって ui=1ui = 1ui=1 となる．

  • DDD の単元全体を D×D^\timesD× と書く．

  • d∈Dd \in Dd∈D が既約元 ⇔defd\xLeftrightarrow{\mathrm{def}} ddef​d は単元や零元ではなく，もし d=pqd = pqd=pq と書けるなら p∈D×p \in D^\timesp∈D× または q∈D×q \in D^\timesq∈D×

  • 整域 DDD が一意分解整域(unique factorization domain, UFD)である

    ⇔def\xLeftrightarrow{\mathrm{def}}def​ 任意の x∈Dx \in Dx∈D について，xxx が単元でも零元でもないなら，既約元 p1,…,pnp_1, \dots, p_np1​,…,pn​ が存在して，x=p1…pnx = p_1 \dots p_nx=p1​…pn​ の形に単元倍の差を除いて一意に書ける．

DDD を整域とする．Q(D)Q(D)Q(D) を DDD の商体とする時，f∈D[X]f \in D[X]f∈D[X] が Q(D)[X]Q(D)[X]Q(D)[X] の元として既約 ⇔D[X]\Leftrightarrow D[X]⇔D[X] の元として既約

RRRを UFDとし，f(X)=anXn+⋯+a1X+a0∈R[X]f(X) = a_n X^n + \dots + a_1 X + a_0 \in R[X]f(X)=an​Xn+⋯+a1​X+a0​∈R[X] とする．次を満たすような RRR の既約元 p∈Rp \in Rp∈R が存在するとする：

  1. p∤anp \nmid a_np∤an​

  2. p∣aj(j=0,…,n−1)p \mathrel{\mid} a_j \quad (j = 0, \dots, n-1)p∣aj​(j=0,…,n−1)

  3. p2∤a0p^2 \nmid a_0p2∤a0​

この時，f(X)f(X)f(X) は RRR および Q(R)Q(R)Q(R)上既約である．

Proof. 補題より fff が RRR 上既約であれば Q(R)Q(R)Q(R) 上でも既約となるので，RRR 上既約であることを示せば十分である．

そこで，fff が RRR 上可約であるとして矛盾を導く．f=gh(g,h∈R[X]∖R[X]×)f = gh \quad (g, h \in R[X] \setminus R[X]^\times)f=gh(g,h∈R[X]∖R[X]×) とする．g=bmXm+⋯+b0,h=cn−mXℓ+…c0g = b_mX^m + \dots + b_0, h = c_{n-m} X^\ell + \dots c_0g=bm​Xm+⋯+b0​,h=cn−m​Xℓ+…c0​ とおこう．条件より m<nm < nm<n でなくてはならない．このとき，p∣a0=b0c0p \mid a_0 = b_0 c_0p∣a0​=b0​c0​ より既約元の基本性質から p∣b0p \mid b_0p∣b0​ または p∣c0p \mid c_0p∣c0​ の少なくとも一方が成立する．また，条件333よりp2∤b0c0p^2 \nmid b_0 c_0p2∤b0​c0​ なので，b0b_0b0​ か c0c_0c0​ のどちらか一方のみが ppp で割り切れることがわかる．よって，p∣b0p \mid b_0p∣b0​ としても一般性を失わない．以下，各 k≤mk \leq mk≤m について p∣bkp \mid b_kp∣bk​ を帰納法により示す．

i<ki < ki<k について p∣bip \mid b_ip∣bi​ が成立するとする．k≤m<nk \leq m < nk≤m<n なので，仮定から ak=bkc0+bk−1c1+⋯+b0cka_k = b_k c_0 + b_{k-1} c_1 + \dots + b_0 c_kak​=bk​c0​+bk−1​c1​+⋯+b0​ck​ は ppp で割り切れる．この時，帰納法の仮定により第二項目以降はすべて ppp で割り切れる．最初の仮定から p∤c0p \nmid c_0p∤c0​ なので，p∣bkp \mid b_kp∣bk​ でなくてはならない．よって帰納法により k≤mk \leq mk≤m について p∣bkp \mid b_kp∣bk​が成立することがわかった．

さて，この時先頭項係数は an=bmcn−ma_n = b_m c_{n - m}an​=bm​cn−m​ である．上の議論から p∣bmp \mid b_mp∣bm​ であるので p∣anp \mid a_np∣an​ となるが，これは仮定に反する．よってfff は RRR 上既約である．

これを応用して多変数多項式の既約判定をしてみよう．

簡単な例ではありますが，f(x,y)=x2+2y∈Q[x,y]f(x, y) = x^2 + 2y \in \mathbb{Q}[x, y]f(x,y)=x2+2y∈Q[x,y] を考えてみましょう．これは Q[x,y]=(Q[y])[x]\mathbb{Q}[x, y] = (\mathbb{Q}[y])[x]Q[x,y]=(Q[y])[x] と思えば，Q[y]\mathbb{Q}[y]Q[y] は UFD ですから一般化 Eisenstein 判定法が使えそうです．そこで，p=yp = yp=y とおいてみれば，y∤1,y∣2y,y2∤2yy \nmid 1, y \mid 2y, y^2 \nmid 2yy∤1,y∣2y,y2∤2y なので前提条件を満たします．よって fff は既約となります．
