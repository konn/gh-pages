Measure Problem

確率論は，全空間が測度 111 を持ち，σ\sigmaσ-加法的であるような測度空間を研究する分野であると云える．そのような確率測度の基本的な例の一つに，単位区間 [0,1][0, 1][0,1] 上の Lebesgue 測度がある．

Lebesgue は当初，R\mathbb{R}R の部分集合は全て Lebesgue 可測であろうと考えていた．より厳密に，Lebesgue 測度は平行移動不変性を保ったまま，P(R)\mathop{\mathcal{P}}(\mathbb{R})P(R) 上に拡張出来るだろうと考えていた．しかし，選択公理を仮定すると Lebesgue 可測でない実数の集合が存在することが，Vitali により示された．

では，平行移動不変性の条件を緩めることで [0,1][0, 1][0,1] の任意の部分集合が可測となるような Lebesgue 測度の拡張を定義出来ないだろうか？この問題を一般化した物が Banach の Measure Problem である：

次を満たすような P(S)\mathop{\mathcal{P}}(S)P(S) 上の確率測度 μ\muμ が定義出来るような無限集合 SSS は存在するか？

  1. μ(∅)=0,μ(S)=1\mu(\emptyset) = 0, \mu(S) = 1μ(∅)=0,μ(S)=1

  2. X⊆Y⇒μ(X)≤μ(Y)X \subseteq Y \Rightarrow \mu(X) \leq \mu(Y)X⊆Y⇒μ(X)≤μ(Y)

  3. 非自明性．(∀x∈S) μ({x})=0(\forall x \in S)\, \mu(\{x\}) = 0(∀x∈S)μ({x})=0

  4. σ\sigmaσ-加法性．Xn (n<ω)X_n \, (n < \omega)Xn​(n<ω) が互いに交わらないなら， μ(⋃n<ωXn)=∑n<ωμ(Xn).\mu\left(\bigcup_{n < \omega} X_n\right) = \sum_{n < \omega} \mu(X_n).μ(n<ω⋃​Xn​)=n<ω∑​μ(Xn​).

以下で測度と云った場合，上の条件のうちで非自明性以外は全て満たしているものとする．また，集合 SSS について P(S)\mathop{\mathcal{P}}(S)P(S) 上で定義された測度の事を単に SSS 上の測度と呼ぶ．

このレポートでは，上の Measure Problem から派生した集合論の問題について簡単に紹介を行う．

ここで，CH の下では ℵ1=2ℵ0\aleph_1 = 2^{\aleph_0}ℵ1​=2ℵ0​ であり，ℵ1\aleph_1ℵ1​ 上に Lebesgue 測度の拡張が存在すれば，ℵ1\aleph_1ℵ1​ は必然的に非自明な測度を持つ最小の基数となり，系 111 より実数値可測基数となることに注意しよう．

以上の結果を連続体仮説に否定的な命題と見るのか，Measure Problem に否定的な命題と見るのかは個人によるだろう．しかし，近年の集合論研究によれば，自然な仮定の下で 2ℵ0=ℵ22^{\aleph_0} = \aleph_22ℵ0​=ℵ2​ となるという結果が幾つか知られており，その意味で（ZFC からは真偽を決定出来ないが）連続体仮説は偽であろうと考える研究者もいる．そうした観点からは，この結果は連続体仮説に否定的な印象を与えるものであると云える．
