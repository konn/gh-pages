簡単な説明

普通の役職構成

  1. Village.new(StandardProfs)で初期化
  2. Village#entry(user, char, prof)で村民をエントリー
  3. Village#startで村開始
  4. Village#updateで更新
    • 返値はHash
      :votes
      投票情報
      :hanged
      処刑
      :medium
      霊能結果
      :fortune

      占い結果のHash

      Keyは占い師、Valueは[占い先, 占い結果]

    • 生存者はVillage#alivers、日数はVillage#progressで確認できます
  5. Village#finished?がtrueを返せば終了

その他

ハムスターを入れたければ、1. のStandardProfsをHamsterProfsに、憑依を有りにしたければPossessionProfsに変更すればOK。
