# 概要とメインの資料

2024年0A月0C日（10月12日）に$\mathrm{ZF}+\mathrm{DC}$ + 任意の実数の集合がLebesgue可測」が成り立つ集合論のモデルである**Solovayモデル**について[alg-dチャンネル](https://www.youtube.com/@alg-dx)で話した時の資料です。
